A l'ouverture du programme :
  -> Lire le fichier de configuration
  -> Si aucun paramètres de connexion LOGIN/PASSWORD
     -> saisir les paramètres de connexion
  -> Si aucune base de données n'est inscrite ou si la base de données n'est pas disponible
     -> créer une nouvelle base de données OU sélectionner une base de données existante
  -> Ouvrir la base de données
